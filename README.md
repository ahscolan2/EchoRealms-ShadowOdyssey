## 🕹️ **Controls & Gameplay**

### ⚡ **Controls:**
- **WASD** or **Arrow Keys**: Move and Jump
- **X**: Echo Phase-Dash (invulnerable dash through enemies)
- **C**: Time Slow (bullet-time mode for precise platforming)
- **Z**: Ground Slam (devastating aerial attack)
- **R**: Restart level if stuck
- **P/ESC**: Pause game
- **F11**: Toggle fullscreen

### 🎯 **Objectives:**
- 🔮 Collect **red artifacts** to progress to the next biome
- 💎 Gather **orbs, keys, and crystals** for points and upgrades
- ⚔️ Defeat enemies using strategic ability combinations
- 🏆 Master all 4 biomes to achieve ultimate victory

### 💡 **Pro Tips:**
- **Triple Jump** gets weaker with each jump - plan your routes carefully
- **Echo Dash** grants temporary invulnerability - use it to pass through enemies
- **Time Slow** is perfect for precise platforming and dodging attacks
- **Ground Slam** deals area damage and creates satisfying screen shake
- Look for hidden collectibles and secret areas
- Energy regenerates over time - manage your abilities wisely

## ⚡ **Advanced Abilities**

### 🦘 **Triple Jump**
- Jump up to **3 times** in the air
- Each subsequent jump is weaker
- Perfect for reaching high platforms

### ⚡ **Echo Phase-Dash**
- **Teleport** through enemies and obstacles
- **Temporary invulnerability** during dash
- **Cooldown-based** - use strategically

### ⏰ **Time Slow**
- Activate **bullet-time** mode
- Slow down physics for precise movements
- **Energy-consuming** but incredibly powerful

### 💥 **Ground Slam**
- Devastating **aerial attack**
- **Area damage** to nearby enemies
- Creates **screen shake** and particle effects

## 🐛 **Troubleshooting**

### Game Won't Load?
1. **Check Browser Console** (F12) for error messages
2. **Try a different browser** (Chrome, Firefox, Safari, Edge)
3. **Clear browser cache** and reload
4. **Make sure JavaScript is enabled**
5. **Try incognito/private mode** to disable extensions

### Performance Issues?
1. **Close other browser tabs** to free up memory
2. **Update your browser** to the latest version
3. **Try lowering browser zoom** if game appears laggy
4. **Check if hardware acceleration is enabled**

### Controls Not Working?
1. **Click inside the game area** to focus
2. **Check if browser shortcuts are conflicting**
3. **Try refreshing the page**
4. **Make sure caps lock isn't interfering**
