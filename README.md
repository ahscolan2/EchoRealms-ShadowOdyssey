# 🌌 EchoRealms: Shadow Odyssey

*A sophisticated 2D platformer with advanced mechanics and cinematic presentation*

## 🎮 How to Play

### ⚡ Quick Start

1. **Clone or Download** this repository
2. **Open `index.html`** in any modern web browser
3. **Start Playing!** - No setup required

**OR**

**🌐 [Play Online](https://ahscolan2.github.io/EchoRealms-ShadowOdyssey/)**

### 🕹️ Controls

- **WASD** or **Arrow Keys**: Move and Jump
- **X**: Echo Phase-Dash (invulnerable dash)
- **C**: Time Slow (bullet-time mode)
- **Z** or **Down + Jump**: Ground Slam
- **R**: Restart level if stuck
- **P/ESC**: Pause game

### 🎯 Objective

**Master your shadow abilities and conquer 4 unique biomes!**

- 🔮 Collect **artifacts** to progress to the next biome
- 💎 Gather **orbs and crystals** for points
- ⚔️ Defeat enemies using strategic ability combinations
- 🏆 Master all biomes to achieve ultimate victory

## 🚀 Game Features

### ✅ Complete Gameplay Experience

- **🏃‍♂️ Advanced Movement**: WASD/Arrow controls with smooth acceleration
- **🦘 Triple Jump**: Jump up to 3 times in air with decreasing power
- **⚡ Echo Dash**: Invulnerable dash with particle trails
- **⏰ Time Slow**: Bullet-time mode with visual overlay
- **💥 Ground Slam**: Devastating aerial attack with screen shake
- **🎨 Rich Visual Effects**: Particle systems for all actions
- **🤖 Smart AI**: Multiple enemy types with different behaviors
- **🏆 Win/Lose System**: Complete victory conditions and restart
- **💚 Health System**: Visual health with invulnerability frames
- **🎒 Inventory**: Live score and collectible tracking

### 🌍 Four Epic Biomes

1. **🌌 Shadow Realm** - Mysterious floating islands with shadow wraiths
2. **🌲 Forest Kingdom** - Vertical platforming with forest guardians
3. **🔮 Crystal Caves** - Precision challenges with crystal formations
4. **⚔️ Boss Arena** - Ultimate multi-level boss battle

### ⚡ Advanced Abilities

- **Triple Jump**: Chain up to 3 jumps with decreasing height
- **Echo Phase-Dash**: Teleport through enemies with invulnerability
- **Time Slow**: Bullet-time for precise platforming and combat
- **Ground Slam**: Area damage attack with screen shake effects

## 💡 Pro Tips

- Each triple jump is weaker - plan your routes carefully
- Echo dash grants temporary invulnerability - use it to pass through enemies
- Time slow is perfect for precise platforming sections
- Ground slam deals area damage and creates satisfying screen shake
- Look for hidden areas and secret collectibles

## 🏃 Quick Development Setup

### Local Server (Recommended)

```bash

git clone https://github.com/ahscolan2/EchoRealms-ShadowOdyssey.git
cd EchoRealms-ShadowOdyssey

# Start a local server (choose one):

```

## 🐛 Troubleshooting

### Game Won't Load?

1. **Check Browser Console** (F12) for error messages
2. **Try a different browser** (Chrome, Firefox, Safari, Edge)
3. **Use a local server** instead of file:// URLs
4. **Clear browser cache** and reload
5. **Check internet connection** (Phaser.js loads from CDN)

### Performance Issues?

1. **Close other browser tabs** to free up memory
2. **Lower browser zoom** if game appears too large
3. **Update your browser** to the latest version
4. **Try incognito/private mode** to disable extensions
